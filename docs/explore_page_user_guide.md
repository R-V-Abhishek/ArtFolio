```

Engagement = (Likes Ã— 1) + (Comments Ã— 3) + (Shares Ã— 5) + (Kudos Ã— 4) + (Views Ã— 0.1)
```

### Why Different Weights?
- **Shares (5 pts)**: Highest value - people recommend it
